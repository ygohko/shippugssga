= SHIPPU GSS =
SHIPPU GSSはタイムアタックシューティングです。以下のような特徴があります:
 * 敵キャラクタの行動パターンはそれなりに豊富ですが、見た目は全部同じなので大変です
 * フルPythonでジェネレータを濫用して安楽に記述されており、貴重なCPU資源を無駄遣いします

= 実行方法 =
Windowsの場合はgss.exeを実行します。
その他のOSの場合はPython、Pygameをインストール後、gss.pyを実行します。

= ルール =
自機を操作し、敵を破壊します。最後に出現するボスを破壊するとゲームクリアとなり、
その早さを競います。

自機が破壊されると残機が1つ減ります。残機が0の状態で破壊されるとペナルティとして
ラップタイムに1分加算されます。

右上にスピード表示があり、自機が画面右側にいる時に加速し、画面左側にいる時に
減速します。スピードが速くなると難易度が上昇しますがその分ラップタイムを短縮できます。
スピードが遅くなると難易度は下降しますがよいラップタイムを得ることができません。

= 操作方法 =
キーボードまたはジョイスティックを使用します。
 * カーソルキー / ジョイスティック方向ボタン ... 自機の移動
 * Zキー / ジョイスティックボタン1 ... ショット(秒間15連射)
 * Xキー / ジョイスティックボタン2 ... ショット(秒間30連射)
 * Escキー ... ゲーム終了 / タイトル画面に戻る
